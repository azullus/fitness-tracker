# 4-Day Strength Training Program  
**Equipment Used:** Squat rack + pull-up bar, adjustable bench, dumbbells  
**Goal:** Strength (low-rep compound focus)

---

## **Weekly Schedule**
| Monday | Tuesday | Thursday | Friday |
|--------|----------|-----------|---------|
| **Lower – Strength** | **Upper – Strength** | **Lower – Volume/Speed** | **Upper – Volume/Speed** |

---

# **DAY 1 — LOWER (Strength)**  
### 1. **Back Squat**  
- 4×3–5  

### 2. **Dumbbell Romanian Deadlift**  
- 4×6–8  

### 3. **Walking Lunges**  
- 3×8 each leg  

### 4. **Calf Raises**  
- 3×12–15  

### 5. **Core**  
- Plank — 3×45 sec  
**or**  
- Weighted Crunch — 3×12  

---

# **DAY 2 — UPPER (Strength)**  
### 1. **Dumbbell Bench Press (Flat or Slight Incline)**  
- 4×3–5  

### 2. **Pull-Ups / Chin-Ups**  
- 4×AMRAP  

### 3. **One-Arm Dumbbell Row**  
- 3×6–8 per arm  

### 4. **Dumbbell Shoulder Press**  
- 3×5–7  

### 5. **Optional Arms**  
- Dumbbell Curls — 2×8–10  
- Triceps Overhead Extension — 2×8–10  

---

# **DAY 3 — LOWER (Volume + Speed)**  
### 1. **Front Squat or Goblet Squat**  
- 4×5–6  

### 2. **Dumbbell Deadlift**  
- 4×5–6  

### 3. **Bulgarian Split Squat**  
- 3×6–8 each leg  

### 4. **Hip Thrust (Dumbbell on hips)**  
- 3×8–10  

### 5. **Core**  
- Side Planks — 3×30 sec per side  

---

# **DAY 4 — UPPER (Volume + Speed)**  
### 1. **Incline Dumbbell Bench Press**  
- 4×5–7  

### 2. **Pendlay-Style Dumbbell Row (Explosive)**  
- 4×5–7  

### 3. **Push-Ups (Weighted if possible)**  
- 3×AMRAP  

### 4. **Lateral Raises**  
- 3×10–12  

### 5. **Optional Arms**  
- Hammer Curls — 2×10  
- Triceps Kickbacks — 2×10  

---

# **Progression Rules**
- Add **2.5–5 lb per week** to squats, Romanian deadlifts, and bench if reps are clean.  
- For pull-ups: add **+1 rep weekly**, then add weight once you hit 10+.  
- For accessories: increase reps first → then add weight.

---

# **Notes**
- Rest **2–3 minutes** between heavy compound sets.  
- Rest **60–90 seconds** for accessory work.  
- Deload every **6–8 weeks** if progress stalls.
